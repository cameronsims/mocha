Mocha - Structs & Classes
PAIR
// pair struct example
mocha::pair<char, float> p;
p['c'] = 12.5f;
The Pair struct is like the std::pair of the standard library.
However, it is also called "mocha::tuple" as a retcon from another struct from the standard library of the same name.
TIMER
// Timer struct example
float get_time() {     mocha::Timer timer(false);
    float time;
    {
        timer.reset();
        time = timer.get();
    }
    return time;
}
The Timer struct is like the a Timing class used for debugging and checking time passed.
MAP
// map struct example
mocha::map<const char*, unsigned int> m(5);
m["data"] = 256;
The Map struct is like the std::map of the standard library
It is not dynamic memory, instead it is static memory.
VECTOR
// vector example
mocha::vector<int> vec;
vec = { 0, 1, 2 };
vec.push(3);
The Vector struct is used as a dynamic array, it can be changed during runtime
VECTOR2 and VECTOR3
Not to be confused with the dynamic array
// vector 2 & 3 example
mocha::vector2 v2(0.0f, 100.0f);
mocha::vector3 v3(0.0f, 100.0f, 200.f);
The Vector struct is used to represent 2D and 3D positions.
